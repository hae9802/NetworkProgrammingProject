게임 방법 : 
-> 스페이스를 눌러 목표에 가까워짐 
-> 일정 목표에 도달하면 키 입력 활성화
-> 활성화 된 키를 누르면 포인트 증가 or 게임 승리

개발 방향 : 
-> 네트워크는 room 개념의 매칭
-> 각자의 도달한 거리를 매칭 된 클라이언트 끼리 알 수 있음 (bar를 이용하여 도달한 거리 표시)
-> 게임은 2D로 가까워지는 느낌을 주는 방법은 원근법 (오브젝트의 크기를 점점 키움, 바닥이 움직임)
-> 게임의 목표에 맞게 어느 위치에 도달하면 키를 활성화 할 것인지 생각하기
-> 게임 오버의 상황을 어떻게 할 것인지 생각하기 (특정 상태에서 움직이면 게임오버 or 다시 처음으로)
-> 무궁화 꽃이 피었습니다 텍스트를 출력하는 주기는 서버에서 제어 (난수 동일화)
-> ETC

현재 개발 상황 :
-> 키를 입력하면 목표가 커짐 (가까워짐)
