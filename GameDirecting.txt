게임은 보트 게임 (장애물을 피해서 빠른 시간에 도달하는 게임)
게임 방법 : 
-> 방향키를 눌러 장애물을 피함
-> 장애물에 부딪히면 체력이 깎임 or 속도가 느려짐
-> 최종 목표에 도달하면 게임 종료 (시간이 더 빠른 사람이 승리) or 상대가 게임 오버가 되면 승리

개발 방향 : 
-> 네트워크는 room 개념의 매칭
-> 각자의 도달한 거리를 매칭 된 클라이언트 끼리 알 수 있음 (bar를 이용하여 도달한 거리 표시)
-> 플레이어는 가만히 있고 오브젝트들이 내려오면서 점점 커짐 (원근법 이용)
-> 장애물을 어떻게 배치할것이고 어떻게 다가오게 할 것인지 생각
-> 플레이어의 체력을 둘 것인지 아니면 속도만 줄여서 최종 목표까지 가게 할 것인지 생각
-> 장애물의 등장 위치는 서버를 통해 모두 동일하게 만들기 (난수 동일)
-> ETC

현재 개발 상황 :
-> 전반적인 게임 완성도 80%
-> 네트워크는 진행 x

-> 게임 서버 구현
-> 네트워크 통신 방식 구상 후 구현
-> UI 구현
-> DB 만들기
